#### noExt

  * Default: true

  * Description: display filename in menu without extention.

  * Example:
      
        -- NO.1 --------------------------------------------------------
        noExt: true

        /Button/Pattern1.js
        ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓
        Button
          - Pattern1
          
        -- NO.2 --------------------------------------------------------
        noExt: false

        /Button/Pattern1.js
        ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓
        Button
          - Pattern1.js
